Welcome to the Monok documentation!
==================================================

Monok_ is an AI service for autogenerated content. This documentation goes through some but not all of its features, if you are a developer, the API documentation is available here. If you are a publisher you will find our `wordpress plugin`_ rather handy.

.. _Monok: https://www.monok.com
.. _wordpress plugin: https://monok.readthedocs.io/en/latest/integrations.html#wordpress

Features
--------

- Autogenerated Unique Text Articles
- Autogenerated Videos with voice narration and animated media
- Automatic Photo Matching with relevant free license media for commercial use
- Relevant Social Media Embeds (Youtube, Twitter, Instagram etc)
